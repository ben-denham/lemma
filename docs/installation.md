# Installation & Usage

You can install the latest version of Lemma in any Python environment
with Pip:

```
pip install lemma
```

## Jupyter Notebooks

If you want to use Lemma from a Jupyter/IPython notebook, you can
install Lemma directly from a notebook cell using the `%pip` magic:

```
%pip install lemma
```
